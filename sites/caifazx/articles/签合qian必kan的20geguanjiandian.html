合同审查要点：签合同前必看的20个关键点

合同审查要点：签合同前必看的20个关键点

"合同签了才发现有陷阱，但已经来不及了。"

"合同条款模棱两可，最后打官司才知道自己吃亏了。"

"对方违约了，但合同里没写清楚怎么处理。"

合同是商业活动的"宪法"，签得好不好，直接关系到企业的利益。但很多企业签合同时不重视，等出了问题才发现合同条款对自己不利。

今天说说合同审查的那些关键点。

一、合同主体的审查

1. 对方主体是否真实存在？

签订合同前，要核实对方公司是否真实存在、是否正常经营。

查询方法：

  • 国家企业信用信息公示系统：查公司基本信息、股东、经营状态
  • 天眼查、企查查：综合查询公司背景、涉诉情况
  • 中国执行信息公开网：查是否为失信被执行人

2. 对方的资质是否合格？

特殊行业需要特殊资质。比如：

  • 建设工程需要建筑资质
  • 食品经营需要食品经营许可证
  • 劳务派遣需要劳务派遣经营许可证

3. 签约人是否有代理权限？

签约人如果不是公司法定代表人，需要核查：

  • 是否有授权委托书
  • 授权范围和期限
  • 授权书是否加盖公章

二、合同标的的审查

4. 标的描述是否清楚？

合同标的（货物、服务、工程等）的描述要具体、明确，包括：

  • 名称、规格、型号
  • 数量、计量单位
  • 质量标准、验收标准

模糊的描述是纠纷的隐患。

5. 标的数量是否准确？

数量要明确，最好用大写数字，防止篡改。要约定溢短装条款（即允许一定幅度的增减）。

6. 质量标准是否明确？

质量标准可以约定：

  • 国家或行业标准
  • 企业标准
  • 双方约定的技术参数
  • 提供样品封存

三、合同价款的审查

7. 价款是否明确？

价款要写清楚：

  • 单价、总价
  • 是否含税
  • 币种（涉及外汇要明确汇率）
  • 支付方式（一次性、分期、按进度）

8. 付款条件是否合理？

常见付款方式：

  • 先付款后发货
  • 先发货后付款
  • 分期付款
  • 按进度付款

要评估对方的信用和履约能力，设置合理的付款条件。

9. 发票条款是否约定？

要明确：

  • 开票类型（增值税专用发票/普通发票）
  • 开票税率
  • 开票时间
  • 不开票的违约责任

四、合同履行的审查

10. 履行地点是否明确？

履行地点涉及：

  • 货物运输费用承担
  • 风险转移时间
  • 管辖法院确定

11. 履行期限是否清楚？

履行期限要具体：

  • 开始时间
  • 完成时间
  • 里程碑节点
  • 逾期违约责任

12. 履行方式是否具体？

明确：

  • 交货方式（送货、自提、物流）
  • 验收方式（数量验收、质量验收）
  • 服务方式（上门服务、远程服务）

五、违约责任的审查

13. 违约情形是否全面？

常见违约情形：

  • 逾期付款
  • 逾期交货/逾期服务
  • 质量不合格
  • 违反保密义务
  • 违反竞业限制

14. 违约责任是否对等？

违约金要合理，不要过高或过低：

  • 过高：可能被法院调低
  • 过低：不足以弥补损失

双方违约责任要对等，不能只约束一方。

15. 违约金计算方式是否明确？

明确：

  • 计算基数（合同总额、欠款金额等）
  • 计算比例（日万分之五、年百分之十等）
  • 上限封顶

六、争议解决的审查

16. 管辖法院/仲裁机构是否明确？

约定管辖要注意：

  • 不能违反级别管辖和专属管辖
  • 约定自己所在地法院/仲裁机构更有利
  • 仲裁一裁终局，效率高；诉讼两审终审，更规范

17. 法律适用是否明确？

国内合同默认适用中国法律，如需适用境外法律，要明确约定。

七、特殊条款的审查

18. 保密条款是否必要？

商业合作中，保密条款很重要。要明确：

  • 保密信息的范围
  • 保密义务的期限
  • 违约责任

19. 知识产权条款是否明确？

涉及技术开发、创意设计等内容，要明确：

  • 知识产权归属
  • 授权使用范围
  • 侵权责任

20. 不可抗力条款是否完善？

不可抗力包括：

  • 自然灾害（地震、洪水等）
  • 社会事件（战争、疫情等）
  • 政府行为（政策变化、征用等）

要明确：

  • 不可抗力的定义
  • 通知义务
  • 证明文件
  • 后果处理

合同审查清单

签约前，逐项检查以下内容：

主体审查：

  • [ ] 公司真实存在
  • [ ] 资质符合要求
  • [ ] 签约人有代理权

条款审查：

  • [ ] 标的描述清楚
  • [ ] 数量准确
  • [ ] 质量标准明确
  • [ ] 价款清晰
  • [ ] 付款条件合理
  • [ ] 履行地点明确
  • [ ] 履行期限清楚
  • [ ] 违约责任对等
  • [ ] 争议解决约定明确

其他审查：

  • [ ] 保密条款（如需要）
  • [ ] 知识产权条款（如需要）
  • [ ] 不可抗力条款
  • [ ] 合同生效条件（如需要审批、盖章等）

签订合同的注意事项

1. 用公司公章，不要用合同专用章以外的章

公章代表公司意志，合同专用章效力等同。部门章、项目章效力有限。

2. 对方盖章要"骑年盖月"

印章要完整、清晰，避免模糊或残缺。不要只盖半个章。

3. 合同份数要够

各方各执一份，确保合同内容一致。

4. 保留签约过程的证据

邮件往来、微信记录、会议纪要等，可以作为补充证据。

5. 合同变更要书面

合同签订后的变更，要签订补充协议，不能只用口头约定。

写在最后

合同审查是专业性很强的工作，涉及法律、商业、行业知识。

几点建议：

  1. 建立合同模板库，减少重复劳动
  2. 重要合同找律师审查
  3. 审查合同要系统，不能只看部分条款
  4. 签约要谨慎，签约就要履行
  5. 履约过程中保留证据

希望这篇清单能帮你做好合同审查，避开那些常见的坑。
