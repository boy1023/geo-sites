# GEO信源站

博一GEO信源站系统，6个行业垂直信源站。

## 站点列表
- zhizaozx.cn - 智造在线（制造业）
- jiaopeizx.cn - 教培在线（教育培训）
- xfzaixian.cn - 消费在线（大消费）
- muyingzx.cn - 母婴在线（母婴）
- nongzizx.cn - 农资在线（农业）
- caifazx.cn - 财法在线（财税法务）

## 部署
每个站点的静态文件在 sites/ 目录下，通过 Cloudflare Pages 部署。

## 目录结构
- sites/ - 各站点HTML文件
- templates/ - 构建模板
- assets/ - 静态资源
- build.py - 构建脚本
